# Project Goal

The main goal of this project is to automate the full YouTube Shorts publishing workflow from one app.

## Step 1: Automatic Video Editing

Build a custom automatic video editor for creating YouTube Shorts.

This step is almost done. The editor supports vertical video layout, draggable video and text elements, styling controls, presets, autosave, end sound effects, and FFmpeg export.

## Step 2: AI Metadata Generation

Use a ChatGPT model to write the short video's:

- Title
- Description
- Tags

The model should generate metadata based on the video/topic so the short is ready for upload with strong SEO and audience targeting.

## Step 3: Automated YouTube Upload

Automate uploading directly from the app.

The app should let the user publish or schedule the exported short to YouTube without leaving the editor.

The final workflow should be:

1. Edit/export the short.
2. Generate title, description, and tags.
3. Upload or schedule the video on YouTube from inside the app.
